【精品软件 第2期】 智能翻译播放器——SPlayer

  • 精品软件
  • 2020-05-10

如今电脑上的视频软件可谓是五花八门，但是对于下载到本地的电影或视频，播放效果都不是很好。

有的是铺天盖地的广告，有的甚至都播放不了。

极物君今天就给大家推荐一款小而强悍的播放器——射手影音。

射手影音（即 splayer）是一款专门播放本地视频的软件。

它能支持现在能够见到的绝大多数视频格式。

用了 射手影音 之后，再也不会有打不开的视频啦！

但这都不值一提，它最强大的功能是能够帮你找字幕，更黑科技的是它还能实时翻译！

无广告、还完全免费！

这简直就是业界良心啊！

下面我们就一起看看如何使用这款强大的软件吧！

下载 射手影音

极物君小贴士：

射手影音有两个版本：最新版跟经典版。

最新版支持实时翻译字幕，但是可能会有一些小bug，不过并不影响正常使用。

经典版虽然不支持实时翻译字幕，但是功能经典！

极物君使用 splayer 4年多了，几乎没有遇到过bug，而且常见的电影或者视频，它都能帮你找到合适的字幕。

所以极物君更推荐安装经典版。

经典版官网下载：https://file.splayer.org/SPlayerSetup.exe?SPlayerSetup2437_701266.exe&from=splayer

经典版备份下载：https://pan.baidu.com/s/1pOE3ZAnWSVJN3Omv9NfAQw 提取码：xhns

最新版官方下载：https://cdn.splayer.org/download/SPlayer-Setup-4.9.1.exe

最新版备份下载：https://pan.baidu.com/s/1CA6vnErhycLW3TgqFyKnpw 提取码：s4nf

安装 射手影音

经典版安装

1.双击下载好的经典版安装包，会弹出 权限询问框 ，点击 是

2.点击 我接受

3.选择安装位置，建议选择默认位置，点击下一步

极物君小贴士：

一般我们常用的软件建议直接安装在 C 盘，这样有利于提高软件的运行速度

如果需要将软件安装至D盘或者E盘，可直将 C 改为 D或E

4.点击 安装

5.等待安装完成，点击 完成

完成后会弹出射手影音的网页，将它关闭即可

最新版安装

1.双击下载好的最新版安装包

2.选择安装用户。一般只有一个用户，直接选择下一步

3.选择安装位置，建议默认安装，选择 安装

4.等待安装完成，将 运行 SPlayer 前面的小勾 去掉（不打开 splayer），点击完成

播放视频

如果视频是默认使用 SPlayer 打开的（右下角有如图小图标），直接双击即可播放视频

若视频不是默认使用 Splayer 打开（右下角没有小图标），则需要 右键——打开方式——选择其他应用——选择Splayer.exe——确定

也可以修改视频默认打开方式为 Splayer，修改方式为：

右键——属性——更改

选择 Splayer.exe

下次双击即可使用 Splayer 打开视频

最新版的使用方式与经典版相同，不再赘述

经典版 查找字幕

默认情况下软件会自动查找字幕，只需要在视频中 右键——字幕 ，选择需要加载的字幕即可

若软件没有自动查找，也可以通过在视频中 右键——字幕——智能匹配——智能字幕匹配下载… 下载字幕

最新版 实时翻译字幕

在最新版中，如果打开的视频没有字幕，则会自动启动实时翻译功能

![实时翻译](C:/Users/dj112/Desktop/公众号/文章/精选软件/【精品软件 第2期】 智能翻译播放器——SPlayer/实时翻译.png)

当然，也可以手动重置实时翻译，在视频中 右键——字幕——重新加载翻译结果

小结：

其实播放器中口碑较好的还有 PotPlayer ，但由于 PotPlayer 设置较为复杂，所以极物君更推荐操作简单、功能同样强大的SPlayer。

大家在使用过程中遇到任何问题，都可以和极物君交流讨论哦！
