【精品软件 第3期】 中国版的 Chrome——360极速浏览器

  • 精品软件
  • 2020-05-10

如果你去网上搜什么浏览器最好用，答案大部分都是 Chrome！

Chrome 无疑是大家公认的浏览器之王，它之所以能被无数人青睐，很大原因是：它有着很多功能强大的插件。

但大家都忽略了一个重要的问题，由于某些不可抗力的原因，我们无法直接安装 Chrome 的插件！

那我们就没办法体验到那些好用的插件了吗？

No，No，No！

有了我们今天的主角，我们不用任何手段也能体验到比 Chrome 更好的网上冲浪体验。

它就是360极速浏览器！

360极速浏览器，极物君 称它为 中国版的 Chrome，它不仅可以直接安装所有的 Chrome 插件，而且插件库中还有很多具有中国特色的插件，比如哔哩哔哩助手，微博助手等等。

可能因为 360的口碑一直饱受争议， 也可能因为不是主打产品的原因，360极速浏览器 一直处于一个小众软件的范畴。

但这丝毫不影响它的强大功能！

基础使用

下载 360 极速浏览器

官方下载：http://down.360safe.com/cse/360cse_12.0.1053.0.exe

备份下载：https://pan.baidu.com/s/1dqmP4ipQdGLgJ9KKGW7P9A 提取码：xuix

安装 360 极速浏览器

1.双击下载好的安装包

2.勾选 同意 用户协议， 点击 快速安装

3.弹出 权限询问框 时 ，点击 是

4.选择自己喜欢的皮肤，点击开始使用

5.基础设置

点击右上角的菜单按钮

点击选项

更换搜索引擎：将 搜索中的 360 搜索更改为 Google，勾选 尝试其它搜索引擎选择，并选择百度

在工具栏中取消勾选 显示头像，显示”主页“按钮，显示”收藏夹“按钮，可以使页面更整洁干净

在个人资料中 登录个人账户

极物君小贴士：

登录账户后浏览器会自动保存收藏夹、浏览历史等信息，在其它设备登陆账户后自动同步，适合需要操作多台设备的朋友使用。

插件的使用与安装（以 集装箱 为例）

1.点击右上角彩色小田字

2.在搜索框中输入集装箱，点击搜索

3.点击安装

4.点击添加

5.在右上角能够找到 集装箱 的图标

现在你就可以愉快的上 Google啦，赶快试一试吧！

集装箱

顾名思义，集装箱就是一些小工具的集合。

它包含以下几款插件：

谷歌助手：能让你直接访问 Google 的神器！

网盘助手：大概是最优雅好用的网盘助手了。

优 惠 购 ：以最优惠的价格，把宝贝抱回家。

下载卫士：拒绝高(捆)速(绑)下载。

Project Naptha

Project Naptha 让你能轻松从图片或者PDF中复制文字，而且文字的识别效果惊人

Adblock Plus

Adblock Plus 是一款超强的广告屏蔽软件，它能够帮你屏蔽掉绝大多数网页上的广告，还你一个干净的网页。

SimpRead 简悦

简悦能够帮你重新排版当前网页，删除广告和无关信息，让你能够获得更好的阅读体验。

而且简悦还有一个很好用的功能——稍后阅读，看到喜欢的文章现在却没有时间看？添加到稍后阅读，等有时间了再慢慢看

油猴脚本（Tampermonkey）

油猴脚本 是一款必备的插件神器，它能让你在网页上运行别人编写的脚本，实现各种各样让人意想不到的功能。

比如 百度网盘直链下载

有了它，你甚至可以不用安装百度网盘。

用网盘助手直接生成直链，全速下载网盘文件。

比如 药方

它能够根据关键词、作者、话题、来源等过滤微博；修改版面，让你看真正想看的微博

你还可以自己去搜想要的功能哦，搜索 油猴 + 你想要的功能（比如 油猴 百度云），基本上都能搜到
